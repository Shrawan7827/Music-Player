# 🎵 Music Player Web App

A modern and responsive **Music Player** built using **HTML, CSS, and JavaScript**.
It allows users to play, pause, and navigate songs with a clean UI and smooth experience.

---

## 🚀 Features

* ▶️ Play / Pause music
* ⏭ Next / Previous track
* 📊 Progress bar with real-time updates
* 🎶 Dynamic song details (title & artist)
* 🖼 Album artwork display
* 🌙 Dark Mode / Light Mode toggle
* 📃 Playlist view (song list)
* 🔁 Repeat functionality (optional)

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Custom UI Styling)
* JavaScript (ES6)
